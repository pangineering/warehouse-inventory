## Clone the project
```bash
  git clone https://github.com/pangineering/warehouse-inventory.git
```



## Installation
1. Create a virual environment
```bash
  virtualenv env
  source env/bin/activate
```
2. Install Libraries
```bash
    pip3 install -r requirements.txt
```

## Run Locally
```bash
    python3 manage.py runserver
```
